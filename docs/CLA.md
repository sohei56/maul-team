## 6. Signing this Agreement

This Agreement is signed electronically through the Project's pull
request workflow, managed by the CLA Assistant Lite GitHub Action.

When You open Your first pull request against the Project, a bot
comments on the pull request asking You to agree to this Agreement. To
sign, post the following comment, exactly as written, on that pull
request:

```
I have read the CLA Document and I hereby sign the CLA
```

Your signature is recorded in the `cla-signatures` branch of the
Project's repository, together with Your GitHub username and the pull
request in which You signed. The signature is a one-time action: once
recorded, it covers that Contribution and all of Your future
Contributions to the Project, and You will not be prompted again on
subsequent pull requests.

By signing, You confirm that You have read and agree to the terms of
this Agreement and that You have the authority to do so.
